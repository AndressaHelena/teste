GIT

Este é um Documento de teste de comandos git

Comandos utilizados:

  • git clone endereço-do-repositório:

    clona o repositório para um diretorio local
  • git branch nome-do-branch:

    cria um branch do repositório
  • touch "nome e extensão do arquivo":

    cria um arquivo, exemplo: touch index.html
  • git add -A :

    adiciona às alterações à fila de commit
  • git commit -m "mensagem de commit":

    commit os arquivos ao repositório
  • git push --set-upstream origin nome-do-branch:

    faz um push das alterações para a branch atual
