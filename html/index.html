GIT teste

Este é um Documento de teste de comandos git

Comandos utilizados

  • git clone : clona o repositório para um diretorio local
  • git branch : cria um branch do repositório
  • touch : cria um arquivo, exemplo: touch index.html
  • git add -A : adiciona às alterações à fila de commit
  • git commit -m "mensagem de commit"
  • git push --set-upstream origin : faz um push das alterações para a branch atual
